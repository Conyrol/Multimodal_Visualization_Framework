# Multimodal_Visualization_Framework

> 黑客马拉松作品：时域语言定位可视化框架
> Youth U md Team by BoJack, Conyrol, QnyKo



## 痛点

1. 科研项目中多关注各种指标，忽略了科研本身的意义
2. 目前不存在多模态相关方面的可视化项目
3. 总而言之，我们的框架为您提供了环境检查、数据检查、结果可视化、数据分析等功能。



## 架构

前端：VUE.js

后端：Django

自然语言处理：Skip-Thought、GloVe

视频图像处理：C3D、I3D

测试模型：TALL、MAC、RWM
